# Welcome to Twico

Twico is a desktop app that reads your unanswered Slack messages and replies in
your voice while you're away. It posts from **your account** — not as a bot —
using the same session your Slack desktop client is logged in with.

This documentation covers what each screen does, how the twin decides whether
to reply, and how to teach it about you (voice + knowledge).
